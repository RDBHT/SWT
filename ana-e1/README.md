# ANA-E1 — Checkliste und Analyse

Einsendeaufgabe ANA-E1 (Softwaretechnik, Kapitel 3.25).

## Projektgegenstand

Konzeptueller Prototyp eines **internen AI-Wissensportals** für eine mittelständische Wohnungsbaugesellschaft (WBG). Mitarbeiter sollen interne Dokumente (Verträge, Hausordnungen, Beschlüsse, Prozessdokumentation, Vorlagen) per natürlichsprachlicher Suche abfragen können. Mittelfristig denkbar als Integration in eine bestehende SharePoint-Umgebung.
